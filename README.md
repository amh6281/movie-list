## What to do?

- useState를 통해 컴포넌트의 상태(동적인 값)를 관리한다.

- Props를 통해 상위 컴포넌트에서 하위 컴포넌트에게 값을 전달한다.

- state를 변경하면 모든 코드들은 다시 실행된다. 특정 코드를 한 번만 실행시키기 위해 useEffect를 사용한다.

- 페이지 이동을 위해 React Router를 사용한다.
